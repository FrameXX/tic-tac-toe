Tic Tac Toe v0.4.1

This is a Tic Tac Toe game as everyone knows it with a bit of customisation. You can set your own number of symbols in 1 line that a player needs to win (win combo) and you can also play against more than 1 player, which can be either another person or a computer.

Controls

The bottom bar shows the current state of the game. It tells you who is playing, who has won, who should confirm their turn, etc. You can place your glyph on the board by simply tapping or clicking on the cell you have chosen. By default, touch screens require you to confirm your moves, but you can change this behaviour in the settings. This is to avoid missed moves where you accidentally tap the wrong cell, which can lead to frustrating experience.

Options

As you may have noticed, there are many ways to customise your experience. There's a brief description under each option. Except for the grid sizes, any changes you make to the configuration will be applied to the game immediately, but I can't guarantee how well they will be applied to the game, so it's always a good idea to restart the game if you make major changes to the configuration. Also note that although you can change the values of some options manually by entering numbers, I cannot guarantee that values that cannot be set normally with the slider bar will work properly with the game. For example, negative values are usually not supported.

Rows count
Number of grid rows.
Columns count
Number of grid columns.
Automatic rows and columns
Optimal rows and columns count will be automatically chosen based on screen size and optimal cell size.
Optimal cell size
Automatic rows and columns count are set according to this value.
Rotate grid
Automatically rotate grid to best fit when window size changes.
Border percentage
Percentage border size in comparison to cell size.
Win combo
Determines how many of same symbols needs to be in one axis for player to win.
Timer minutes
Total minutes, that each human player can spend on his/her turns. When set to 0, timer will be disabled.
Symbols per turn
Determines how many symbols can each player place on the grid per 1 turn.
UI mode
Choose your preferred UI mode. By default the mode should change automatically according to your system settings.
Fat finger
Have you got fat fingers like your dad? Don't worry. This option is here to make sure you really did tap or click where you meant to by asking you to confirm your decision.
Rotate on turn
This option is suitable for 2 players, each playing on a device, but sitting opposite each other. With this option, the game will rotate 180 degrees on each turn instead of the players having to rotate the device.
Sound effects
If allowed different sound effects will be played on some in-game events.
Show cell indexes
Shows row and column index on every cell.
